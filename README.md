# Symfony学習
## 参照
The Fast Track - 基礎から最速で学ぶ Symfony 入門
https://symfony.com/doc/6.2/the-fast-track/ja/index.html

## 手順
### 開始
```
cd プロジェクトルート
symfony server:start -d
symfony open:local
```

### 終了
```
cd プロジェクトルート
symfony server:stop
```
